# L-T-internship-
Analysing the data and insights out of it from the provided Retail dataset


On undergoing 2 month internship under Larsen and Toubro company, I had attempted my data analysis and science insights in a Retail dataset

The above data set contains 8 columns
InvoiceNo	StockCode	Description	Quantity	InvoiceDate	UnitPrice	CustomerID	Country

Each columns are segregated numerically and categorically and first EDA is done, then accordingly ML is used to drive to conclusions about the data provided.
